## Основной стек

**DATABASE**
  - PostgreSQL

**Frontend - ReactJS**
  - MATERIAL UI (UI/UX компоненты)
 - MOBX-STATE-TREE (Модели и состояния)
 - NEXT JS (Серверный рендер и маршрутизация)

**Backend - Python**
  - FlASK (Микросервисы и маршрутизация)
 -  SQLALCHEMY (ORM для работы с БД)
 - PY-KIM (Сериализация моделей)
